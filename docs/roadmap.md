# Product Roadmap

## Now (0-3 miesiące)
- Stabilizacja workflow core (steps + events + approval).
- Rozbudowa compliance o reguły policy per branża.
- Dashboard usage (AI cost + YouTube quota).
- Lepsza observability błędów workflow.

## Next (3-6 miesięcy)
- Integracje natywne (YouTube upload lifecycle, CMS connectors).
- Role-based approvals (multi-level approval).
- Playbooks per typ kanału (edukacja, e-commerce, SaaS).
- Powiadomienia Slack/Email dla blokad compliance.

## Later (6-12 miesięcy)
- Benchmarki performance między kanałami.
- Predictive planning (prognozy kosztów i wyników).
- Marketplace agent templates.
- Enterprise controls (audit exports, advanced policies, region controls).

## Product guardrails
- Human-in-the-loop zostaje obowiązkowy dla publikacji.
- Compliance i cost governance są "always-on".
